# Roadmap

## Highest Priority

- hidden live-session failover
- DHT store admission hardening
- stronger provider and path diversity
- long-run abuse and churn testing
- Windows runtime compatibility without protocol drift

## Medium Priority

- disk-backed large DHT caches
- WAN tuning for lookup fanout and refresh intervals
- better hidden descriptor traffic-analysis resistance

## Lower Priority

- real QUIC transport
- real eBPF/XDP relay acceleration
- polished desktop packaging
