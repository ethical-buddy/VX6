# VX6 Comms Android Workspace

This directory is reserved for Android-native implementation that consumes VX6 protocol via SDK bridge and keeps `main/internal/sdk` independent of UI app code.

See:
- `../vx6comms/ANDROID_DEVELOPER_TASKS.md`

## Scope

- Kotlin + Compose app
- Foreground service node runtime
- Room storage + file/media index
- WebRTC calls with TURN fallback
