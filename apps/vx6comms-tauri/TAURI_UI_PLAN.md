# VX6 Comms Tauri UI Plan

## Core Visual Direction

- Clean, minimal, enterprise-ready interface
- Neutral palette with strong contrast (avoid default blue/purple look)
- Left rail: contacts, presence, unread counters
- Center: conversation and media timeline
- Right utility panel: call controls, diagnostics, device settings

## Phase Plan

1. Shell + navigation
2. Chat + file UI
3. Calls + diagnostics
4. Admin/policy panels
5. Packaging/signing

## Quality Targets

- Fast startup (<2s on baseline hardware)
- Responsive UI at 60fps for typical chat flows
- Accessibility:
  - keyboard navigation
  - readable contrast
  - large-click areas
